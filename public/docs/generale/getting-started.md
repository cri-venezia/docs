## Guida Rapida

Questa sezione fornisce una panoramica su come navigare e utilizzare questo sito di documentazione.

## Navigazione

Usa il menu laterale (o il pulsante "hamburger" ☰ su mobile) per navigare tra le diverse sezioni.

Le sezioni sono organizzate per categoria o nome del plugin (ad esempio, "Generale" o "cri-trasporti").

Clicca sul nome di un plugin (es. "cri-trasporti") per mostrare l'elenco dei file di documentazione disponibili.

Clicca sul nome del file specifico (es. "Installazione" o "Configurazione") per caricare e leggere la guida.
